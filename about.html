About Recipe Finder

Ini adalah aplikasi web sederhana yang memakai api dari TheMealDB.

Features

  • mencari resep.
  • melihat video resep.
  • TheMealDB adalah API yang menyediakan informasi tentang resep makanan dan minuman.
  • Prak10_248 adalah repositori GitHub untuk proyek ini.
  • Team SearchRecipe_PDW_team-project Berisi Team Project original dari Praktikum Pemrograman Web.
